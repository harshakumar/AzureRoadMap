// 02 — Quality Engineering

Study Testing

From manual fundamentals to full automation mastery — ISTQB, Playwright, API testing, and beyond.

Roadmap

01

Testing Foundations (ISTQB Foundation)

SDLC & testing lifecycle, test levels (unit/integration/system/acceptance), test techniques, defect management, and the seven principles of testing.

Beginner 2–3 weeks
02

Manual Testing Techniques

Black-box techniques: equivalence partitioning, boundary value analysis, decision tables, state transition. Test plan & test case design. Bug reporting.

Beginner 3–4 weeks
03

API Testing

REST principles, HTTP methods & status codes, testing with Postman & Newman, contract testing with Pact, schema validation.

Intermediate 3–4 weeks
04

UI Automation (Playwright / Cypress)

Page Object Model, locator strategies, async/await patterns, parallel execution, test fixtures, and CI integration.

Intermediate 5–6 weeks
05

Performance & Security Testing

Load testing with k6 / JMeter, identifying bottlenecks, OWASP top 10 basics, and integrating tests into CI/CD pipelines.

Advanced Ongoing
