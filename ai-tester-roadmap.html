Month 1 — AI Foundations & Prompt Engineering

Understand how LLMs work, why they fail, and start writing prompts that surface bugs

Weeks 1–4~8 hrs/week100% Free
Week 1

AI & ML Fundamentals

  • What is AI, ML, LLMs — plain English, no math
  • How neural networks learn from data
  • Generative vs classification vs recommendation AI
  • AI product lifecycle vs traditional software lifecycle
  • Key terms: training data, model, inference, hallucination, bias, tokens
Week 2

Prompt Engineering

  • Anatomy of a prompt: role, context, instruction, format
  • Zero-shot vs few-shot vs chain-of-thought prompting
  • System prompts vs user prompts — why it matters for testing
  • Prompt injection & jailbreaking concepts
  • Practice: rewrite 20 prompts, compare outputs side by side
Week 3

How LLMs Fail

  • Hallucinations — types and what triggers them
  • Bias in outputs: gender, race, recency bias
  • Context window limits and truncation bugs
  • Non-determinism: same input → different outputs
  • Map each failure mode to your manual testing mental models
Week 4

AI Testing Mindset

  • Traditional testing vs AI testing — the key differences
  • Probabilistic outputs: what does "pass" even mean?
  • Metrics: accuracy, precision, recall, F1 — what testers need to know
  • Write your first 10 AI test cases for a chatbot
  • Read the ISTQB CT-AI syllabus overview (free PDF)

Month 2 — AI Testing Techniques & Tools

The real craft: eval frameworks, Python basics, red teaming, safety testing

Weeks 5–8~10 hrs/week100% Free
Week 5

Functional AI Testing

  • Testing AI search, recommendations, autocomplete features
  • Writing test cases for non-deterministic outputs
  • The oracle problem: how to judge "correct" AI output
  • Building a golden dataset — your reference answer set
  • Equivalence partitioning applied to prompt inputs
Week 6

LLM Evaluation Frameworks

  • Ragas — evaluate RAG pipelines (open source, free)
  • Promptfoo — unit testing discipline for prompts (MIT license)
  • DeepEval — open source LLM eval framework
  • BLEU, ROUGE, LLM-as-a-judge scoring explained
  • Build a simple eval harness in Google Colab (free GPUs)
Week 7

Python for AI Testers

  • Variables, loops, functions — bare minimum to get started
  • Calling OpenAI / Anthropic / Gemini APIs in Python
  • Parsing JSON API responses and writing assertions
  • pytest basics: write 5 LLM test cases with assertions
  • Use Cursor / GitHub Copilot (free tier) to write tests faster
Week 8

Safety & Responsible AI

  • AI safety testing: harmful content, PII leakage, toxicity
  • Bias testing across protected attributes in outputs
  • GDPR / data privacy implications for AI features
  • OWASP LLM Top 10 — all 10 risk categories (free guide)
  • Write a 1-page AI risk test plan for a sample chatbot

Month 3 — Portfolio, Specialisation & Job Prep

Ship real projects, pick a niche, position yourself as an AI QA Engineer

Weeks 9–12~10 hrs/week100% Free
Week 9

AI Test Automation Project

  • Pick a public LLM app: ChatGPT, Claude, Gemini (all free tiers)
  • Write 30+ automated eval tests via pytest + API
  • Cover: accuracy, safety, latency, edge cases, adversarial inputs
  • Publish to GitHub (free) with a quality README
  • Document bugs found with severity classification
Week 10

Pick a Specialisation

  • Path A: Conversational AI / Chatbot testing
  • Path B: Computer Vision / image AI app testing
  • Path C: AI-assisted test automation (Playwright + AI)
  • Find 3 open source tools in your niche and learn them
  • Publish a LinkedIn article on your chosen specialisation
Week 11

Portfolio & Personal Brand

  • GitHub: 2–3 AI testing repos, well documented (free)
  • LinkedIn: reframe 5 years as "AI-ready QA" experience
  • Write a case study: "How I'd test [AI product X]"
  • Contribute to an open source AI testing tool
  • Join Ministry of Testing community (free membership)
Week 12

Interview Prep & Networking

  • Prep top 20 AI testing interview questions (use ChatGPT free)
  • Practice: "How would you test a RAG chatbot?" — mock answer
  • Apply to 5+ AI QA roles — tailor CV with AI keywords
  • Explore ISTQB CT-AI exam (the one investment worth making)
  • Review Anthropic + OpenAI developer docs (both free)

Weekly Milestones
🎯

Week 1 — You can explain how LLMs work to a non-technical colleague

You understand tokens, training, inference, and hallucinations. You can map each to testing risks.

✍️

Week 2 — You've written 20 varied prompts and analyzed all outputs

You spot flaky, biased, or inconsistent AI outputs. Your manual testing instincts are activating on probabilistic systems.

📄

Week 4 — Your first AI test plan exists in writing

10+ test cases for a chatbot. Includes adversarial inputs, context window tests, and consistency checks.

🛠️

Week 6 — You've run a Promptfoo or Ragas evaluation and have a score

You have a numeric quality score for an AI system. You can explain what it means to a developer in a standup.

🐍

Week 7 — You've written Python that calls an LLM API and asserts on the output

5+ automated test cases running with pytest. You've removed the "no code" ceiling from your AI QA career.

🔴

Week 8 — You've red-teamed a public AI app using Garak or manual prompts

OWASP LLM Top 10 is no longer abstract — you've reproduced at least one real vulnerability class in the wild.

🚀

Week 10 — Your GitHub has a real AI testing project with 30+ test cases

Documented README, results logged, bugs found. This is the portfolio piece that gets you interview callbacks.

🏆

Week 12 — Applied to 5+ AI QA roles, ISTQB CT-AI on your radar

Your LinkedIn reflects an AI QA identity. You can answer "how would you test a RAG chatbot?" fluently in any interview.
