Study Azure

Structured path from fundamentals to advanced architecture, plus curated resources for every stage.

Roadmap

01

Cloud Fundamentals (AZ-900)

Core concepts: cloud models (IaaS/PaaS/SaaS), Azure regions, subscriptions, and shared responsibility. Target: pass AZ-900.

Beginner 2–4 weeks
02

Core Services Deep Dive

Compute (VMs, App Services, AKS), Storage (Blob, Files, Tables), Networking (VNets, NSGs, Load Balancers), and Identity (Entra ID).

Intermediate 4–6 weeks
03

Administrator Path (AZ-104)

Manage Azure resources, implement storage & compute solutions, configure virtual networking, and monitor infrastructure.

Intermediate 6–8 weeks
04

Developer / DevOps Path (AZ-204 / AZ-400)

Azure Functions, Cosmos DB, API Management, CI/CD with Azure DevOps, Infrastructure as Code with Bicep & Terraform.

Advanced 8–10 weeks
05

Architecture & Best Practices

Well-Architected Framework, landing zones, cost optimisation, security benchmarks, and real-world design patterns.
